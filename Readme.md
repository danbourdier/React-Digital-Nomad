# Welcome to Digital Nomad! A way to visit the city of your dreams in person from the comfort of home!


## Live Site
https://take-a-walk.netlify.app/


## Instructions
  1. npm install
  2. run `npm run start`

### Technologies used
  1. HTML5
  2. CSS3
  1. JavaScript (ES6)
  2. React.js (Hooks)
  3. Youtube video player API
  3. Youtube Data API
  4. Netlify


### Locations supported
  1. Japan
  2. France
  3. New York City
  4. Germany
  5. India
  6. Italy
