# Welcome to Digital Nomad! A way to visit the city of your dreams in person from the comfort of home!


## Live Site
https://take-a-walk.netlify.app/


## Instructions
  1. npm install
  2. run `npm run start`
  3. Open browser and go to localhost: `XXXXX`
